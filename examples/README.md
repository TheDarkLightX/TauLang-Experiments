# Examples

This directory contains original Tau examples and experiment inputs written for this research repo.

Rules:

- Keep examples small and reproducible.
- Do not copy Tau Language source files here.
- Prefer examples that have a matching trace or proof artifact.
- If an example requires a patched Tau checkout, name the required patch in the example header.

Suggested subdirectories:

```text
examples/finite_tables/
examples/qelim/
examples/recurrence/
examples/nso_splitters/
```
